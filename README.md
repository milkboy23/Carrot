# Carrot README

A Visual Studio Code Extension targeted at students. Makes code comments unobtrusive, formatted and capable of storing multi-media.

## Features

- Unobtrusive: Hide your comments behind a carrot icon

> image or animation

- Formatted: Write your comments in markdown or use the Carrot Note feature for full customization

> image or animation

- Multi-media: Store images or create drawings in the Carrot Note feature.

> image or animation



## Extension Settings

This extension contributes the following settings:

* `myExtension.enable`: Enable/disable this extension.

## Known Issues

The Carrot Comments can only be removed by disabling the extension
